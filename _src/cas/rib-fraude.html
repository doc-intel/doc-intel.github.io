Redirection de virement : l'IBAN retapé d'un RIB

Un fraudeur intercepte un RIB, masque l'IBAN et en tape un autre : le virement part sur son compte. La fraude ne se voit pas à la lecture. Elle est dans le fichier : l'IBAN d'origine y survit.

Le problème

Un gestionnaire reçoit un RIB pour payer un bénéficiaire. Le document paraît normal. Mais l'IBAN a peut-être été remplacé après coup, pour rediriger le paiement. La page imprimée ne dit rien de cette manipulation : la preuve est dans le fichier, pas dessus.

L'approche

Le détecteur lit le document lui-même, pas seulement son texte. Il regarde les métadonnées (logiciel producteur, dates de création et de modification, fuseaux horaires) et l'historique de révisions du PDF. Un enregistrement incrémental peut laisser une version antérieure enfouie dans le fichier — ici, le RIB avec son IBAN d'origine. Chaque signal est circonstanciel seul, donc le verdict est gradué : vert quand rien ne ressort, ambre quand l'indice n'est que circonstanciel, rouge quand le document a été démontrablement édité après création avec l'état antérieur toujours à l'intérieur.

Le fil : une révision enfouie n'est pas une heuristique. Le fichier contient les deux IBAN. On reconstruit celui d'avant retouche et on le lit — la preuve la plus directe d'une redirection de virement.

La méthode, étape par étape

Lire les métadonnéesLogiciel producteur, dates création vs modification, fuseaux — le premier fil à tirer.
Inspecter les révisionsUn enregistrement incrémental laisse une version antérieure enfouie ; la reconstruire donne l'IBAN d'origine.
Graduer le verdictVert (propre), ambre (circonstanciel, à revoir), rouge (édité après création, état antérieur à l'intérieur).
Rester déterministeMétadonnées et révisions se lisent dans le fichier, sans modèle ; la réponse est reproductible.

Ce que ça montre

La fraude au virement se détecte sans base externe : le RIB trafiqué porte les deux IBAN. La détection commence avant tout modèle de langage — la structure du fichier porte la preuve.

Livrables

Un rapport sur deux RIB synthétiques (authentique, IBAN retapé) avec un verdict gradué et les signaux exacts qui se sont déclenchés. Toutes les pièces sont fabriquées.
