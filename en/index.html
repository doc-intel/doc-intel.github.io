The series

Seven volumes, one engine

Volumes 1 to 4 build the engine, layer by layer. Volumes 5 to 7 embed it into three products: a web app, a desktop app, a SaaS.

Live
Vol. 1

PDF RAG

The four bricks (parsing, question, retrieval, generation), from a single PDF to corpus scale. Every answer stays sourced, down to the exact page.

Soon
Vol. 2

Multi-format, multi-intent

Word, Excel, PowerPoint and mail join the PDF, each with its own parser. New intents arrive: translate, summarize, compare, proofread.

Soon
Vol. 3

Production with tools

The document is no longer just read, it is produced: a form filled, a table generated, a file rewritten. A catalog of tools the engine knows how to call.

Soon
Vol. 4

Agentic RAG with audit

An agent picks its actions, remembers what it did, and leaves a readable trail. Glass box: every step is verifiable and replayable.

Soon
Vol. 5

Web app (Streamlit)

The engine behind a data-scientist UI: drop a folder, ask a question, read the sourced answer. Built to prototype fast.

Soon
Vol. 6

Desktop app (Electron)

A local workstation that indexes a scanned folder and answers offline. Python sidecar, data that never leaves the machine.

Soon
Vol. 7

Multi-tenant SaaS

The server edition, multi-tenant: Postgres, S3, OIDC auth, billing. The same engine, run at scale.
