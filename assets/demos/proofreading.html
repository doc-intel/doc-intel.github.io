PROCESSING — what the pipeline does

  1. build a line_df from the inline text (page_num=1, line_num=1..N)
  2. doc_proofread(line_df_override=line_df, mode=grammar, proofread_caller=llm_parse+_PatchList)
  3. for each patch : locate `before` substring in the source line → synthesize start/end offsets
