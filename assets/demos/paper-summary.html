PROCESSING — what the pipeline does

  1. parse_pdf(pdf) → line_df + toc_df
  2. chunk_by_toc(line_df, toc_df) → list of Chunk (one per section)
  3. make_openai_leaf() → summarize each chunk into a SectionSummary with anchored Claims
  4. make_openai_parent() → merge leaves into a root SectionSummary
  5. faithfulness gate → drop any Claim whose source_span_ids don't exist in line_df
