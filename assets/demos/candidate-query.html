Querying the candidate pool in plain English

A recruiter's question turned into a bounded SQL query over the candidate database, executed, with the query shown next to the result.

"Python developers with Spark and 5+ years, based in Paris"
SELECT name, skills, experience, location
FROM candidates
WHERE skills LIKE '%Python%' AND skills LIKE '%Spark%'
  AND experience_years >= 5 AND location = 'Paris';
