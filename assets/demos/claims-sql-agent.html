Agent SQL sinistres

Interroger une base de sinistres propre en langage naturel. Question → SQL → tableau → réponse.

Démo statique : les réponses ci-dessous sont figées (exécutées une fois sur le portefeuille fictif de 600 sinistres). Aucun serveur, aucun appel réseau. Pour l'interactif temps réel, lancez l'app Streamlit dans ../app/. Cliquez une question pour dérouler.
Base fictive : 600 sinistres, 5 assureurs, 8 régions, 5 garanties, 3 devises. Chaque requête générée est validée en lecture seule (SELECT uniquement, une seule instruction, LIMIT imposé).
Démonstration statique (données figées), pas l'outil dynamique. Pour un outil sur vos propres documents, contactez-nous. · Études de cas
